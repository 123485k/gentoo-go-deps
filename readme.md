You can use this workflow generate gentoo deps tarball

Support:

 - Golang: `${P}-deps.tar.xz` and `${P}-vendor.tar.xz`
 - Javascript: `${P}-node_modules.tar.xz`
 - rust: `${P}-crates.tar.xz`
